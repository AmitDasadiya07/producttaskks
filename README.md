# Product Management Module - Laravel

This repository contains a full **Product Management Module** built with **Laravel 12**, **MySQL**, and **Bootstrap 5**.  
It includes **CRUD operations** for products and categories, **live search**, **sorting**, **pagination**, and image gallery handling.

---

## Features

- **Category Management**
  - Add, Edit, Delete categories
  - Slug generation

- **Product Management**
  - Add, Edit, Delete products
  - Assign products to categories
  - Upload multiple images (gallery)
  - Technical specification with CKEditor
  - Optional collection field

- **Live Search & Sorting**
  - Search products by name in real-time
  - Sort by Name ↑/↓ or Newest

- **Pagination**
  - Paginate products with Bootstrap 5 styling
  - Works with live search and sort

- **Responsive Design**
  - Built with Bootstrap 5
  - Fully responsive and mobile-friendly
